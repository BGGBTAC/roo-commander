# Basic Style Guide Examples

Examples of simple style guides that can be generated from a completed design.

## Example 1: Simple Palette & Fonts

```markdown
# Style Guide: [Page Name] Design

## Colors

*   **Primary:** `#005f73` (Dark Blue/Teal) - Used for buttons, key highlights.
*   **Secondary:** `#0a9396` (Medium Teal) - Used for secondary elements, backgrounds.
*   **Accent:** `#ee9b00` (Orange/Gold) - Used for calls-to-action, highlights.
*   **Background:** `#ffffff` (White)
*   **Text:** `#001219` (Very Dark Blue)

## Typography

*   **Heading Font:** Playfair Display (Serif)
    *   H1: 39px, Bold
    *   H2: 31px, Bold
    *   H3: 25px, Bold
*   **Body Font:** Lato (Sans-serif)
    *   Base Size: 16px (1rem)
    *   Line Height: 1.5

```

## Example 2: More Detailed

```markdown
# Style Guide: [Project/Feature Name]

## 1. Color Palette

| Role        | Hex       | Usage                                    |
| :---------- | :-------- | :--------------------------------------- |
| Primary     | `#e63946` | Main actions, branding, key highlights   |
| Secondary   | `#457b9d` | Secondary buttons, info sections         |
| Accent      | `#a8dadc` | Subtle highlights, borders, icons        |
| Background  | `#f1faee` | Main page background                     |
| Surface     | `#ffffff` | Card backgrounds, modals                 |
| Text        | `#1d3557` | Body text, primary headings            |
| Text Muted  | `#6c757d` | Secondary text, meta info              |
| Error       | `#d90429` | Error messages, validation             |

## 2. Typography

### Font Families
*   **Headings:** Montserrat, sans-serif
*   **Body:** Merriweather, serif

### Type Scale (Base: 16px)
*   H1 (`<h2>` in code): 2.5rem (40px), Bold (700)
*   H2 (`<h3>` in code): 2rem (32px), Bold (700)
*   H3 (`<h4>` in code): 1.5rem (24px), Semi-Bold (600)
*   Body (p): 1rem (16px), Regular (400)
*   Small Text: 0.875rem (14px), Regular (400)

### Line Height
*   Body Text: 1.6

## 3. Key Components (Visual Style)

*   **Buttons (Primary):** Background: Primary (`#e63946`), Text: White, Padding: 0.75em 1.5em, Border-radius: 4px. Hover: Slightly darker primary.
*   **Buttons (Secondary):** Background: Secondary (`#457b9d`), Text: White, Padding: 0.75em 1.5em, Border-radius: 4px. Hover: Slightly darker secondary.
*   **Cards:** Background: Surface (`#ffffff`), Border-radius: 8px, Box-shadow: [Define subtle shadow].

```

*(Generate a simple guide like Example 1 by default if requested. A more detailed guide like Example 2 requires more specific extraction or definition.)*
