# Design System

This directory contains documentation related to the project's design system.

## Purpose

The design system provides a collection of reusable components, patterns, and guidelines that ensure consistency and efficiency in UI/UX development.

## Contents

*   **Components:** Documentation for individual UI components (buttons, forms, modals, etc.).
*   **Patterns:** Descriptions of common UI patterns and interactions.
*   **Guidelines:** Principles and best practices for using the design system.
*   **Assets:** Links to or storage for relevant design assets (icons, fonts, etc.).

This context is primarily used by the `design-lead` mode and frontend developers.
